# lwx-magic-code
一个智能代码生成平台，支持用户通过自然语言描述需求自动生成可运行前端项目，可视化编辑页面元素并实时对话优化，一键云端部署与源码下载。
## 如何拉取项目

复制Https链接或者下载zip本地源代码压缩包

```

https://github.com/77lwx/lwx-magic-code.git

```
